# PRD 작업 로그(사용량 추적)

목적: 에이전트/작업자가 PRD를 “어떤 작업에서, 어떤 섹션을 읽고/수정했는지” 남겨서 PRD 의존성과 변경 이력을 깔끔하게 추적한다.

규칙:
- **모든 작업 1회당 1개 엔트리**(짧아도 됨)
- 최소 기록: 날짜/작업/읽은 문서/수정한 문서/결과(산출물)
- 링크/경로는 가능하면 파일명으로 명시(예: `01-ia-sections.md`)
