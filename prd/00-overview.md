# 00) 개요 / 목표 / 타깃 / 메시지

최종 업데이트: 2026-04-02

## 1) 개요

### 1.1 제품/페이지 정의
- 브랜드명: **구찌골프 / GOODGGOLF**
- 형태: 단일 랜딩페이지(필요 시 개인정보처리방침 별도 페이지 1개)
- 목적: 유튜브/인스타 유입 → 상담 → **체험수업 예약** 전환을 최대화한다.
- 핵심 제공가치: 트랙맨 기반 실내 레슨 스튜디오 (필드/숏게임/외부 레슨 포함)
- 시설: 타석 2개, 예약제 전용, 레슨 전문 스튜디오

### 1.2 전환 퍼널
```
유튜브/@good_ggolf  ──┐
인스타/good_ggolf   ──┤→ 랜딩페이지 → 상담(전화/카카오/폼) → 체험수업 → 정규수강
기타 유입            ──┘
```
- **핵심 전환 목표: 체험수업 신청**
- CTA 1순위 카피 방향: “체험수업 신청하기” (단순 “상담문의”보다 구체적 행동 유도)

### 1.3 v1의 핵심
- 전문적인 인상(신뢰/증거) + 즉시 행동(체험수업 CTA)
- v1 CTA(기본): **전화 + 카카오톡 채널 + 문의 폼**
- 코치: 4명(추가/변경 가능한 구조로 구현)

### 1.3 타깃
- Primary: 개인 골퍼(B2C) — 레슨 니즈/실력 개선/필드 적응/숏게임 집중
- Secondary: 법인/단체(B2B) — 단체 레슨/행사/제휴 문의(초기에는 섹션/CTA로 최소 지원)

### 1.4 언어
- 한국어 단일(v1)

## 2) 목표 / 성공 지표 (KPIs)

### 2.1 1차 목표(전환)
- 리드(문의) 수: `전화 클릭 수` + `카카오 클릭 수` + `폼 제출 수`

### 2.2 보조 지표
- CTA 클릭률(페이지 대비): `cta_*_click / page_view`
- 폼 제출률: `form_submit / form_start(또는 form_view)`
- 품질 지표: 유효 리드 비율(운영에서 판단) `valid_lead / total_lead`
- 채널별 전환: UTM 기준 소스/매체별 전환 비교

### 2.3 성능/품질 목표(권장)
- 모바일 우선
- Lighthouse(모바일 기준) 권장 목표:
  - Performance: 80+
  - Accessibility: 90+
  - Best Practices: 90+
  - SEO: 90+
- Core Web Vitals(권장):
  - LCP: 2.5s 이하
  - CLS: 0.1 이하

## 3) 사용자 페르소나 / 상황

### 3.1 개인 골퍼(B2C)
- 상황: 레슨을 받으려는데 “누가/어떻게/어디서/얼마나”가 불명확함
- 니즈: 내 문제(드라이버/아이언/숏게임/필드 멘탈 등)에 맞는 코칭, 신뢰, 일정/장소/비용의 감
- 불안: 사기/과장/효과 없음/가격 불투명/내 실력과 미스매치

### 3.2 법인/단체(B2B)
- 상황: 단체 프로그램/행사/복지 제휴를 빠르게 문의하고 싶음
- 니즈: 가능 범위(인원/지역/형태), 일정 조율 용이, 신뢰할 수 있는 진행력

## 4) 핵심 메시지(카피) 프레임

### 4.1 메시지 구조(Problem → Solution → Differentiation → Evidence → Action)
- Problem: “필드에서 무너지는 스코어 / 숏게임이 안 붙는 고민 / 레슨 효과가 체감되지 않음”
- Solution: “상황별 맞춤 코칭 + 루틴/드릴로 재현성 만들기”
- Differentiation: “필드/숏게임/외부 레슨까지 커버, 실전 중심”
- Evidence: “{{성과/경력/자격/후기/미디어}}”
- Action: “지금 상담받기(전화/카카오/문의 폼)”

### 4.2 v1 히어로 카피 초안(placeholder)
- 메인 헤드라인: `{{HEADLINE_1}}`
- 서브 문장: `{{SUBHEAD_1}}`
- 신뢰 배지(짧게): `{{TRUST_BADGE_1}}` / `{{TRUST_BADGE_2}}` / `{{TRUST_BADGE_3}}`
