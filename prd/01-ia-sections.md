# 01) 정보구조(IA) / 섹션 요구사항

최종 업데이트: 2026-04-02

> 구현은 단일 페이지 스크롤(섹션 앵커) 기준. 모바일에서 CTA 접근성을 최우선으로 한다.

## 5) 정보구조(IA) / 섹션 요구사항

### 5.1 Header
- 요소: 로고/이름, 메뉴(선택: 프로그램/후기/FAQ/위치), CTA 버튼 1개(“상담문의”)
- 모바일: 햄버거 or 최소 메뉴 + CTA

### 5.2 Hero (Above the fold)
- 목표: 5초 안에 “누구/무슨 가치/어떤 행동”을 이해시키기
- 구성:
  - 헤드라인/서브카피
  - 1차 CTA 3개(전화/카카오/문의 폼으로 스크롤 or 모달)
  - 비주얼: 대표 사진/짧은 영상(선택)
- 요구사항:
  - CTA는 동일 톤(“상담문의”)으로 통일하되, 채널만 구분
  - 스크롤 유도(“프로그램 보기”) 2차 CTA는 선택

### 5.3 신뢰 요소(코치 소개/증거)
- 구성(권장):
  - 코치 소개 카드: 이름/한줄 소개/핵심 경력 3개/사진
  - 자격/경력/성과(가능한 범위에서 정량/구체)
  - 후기/사례(가능하면 실명 비노출 형태)
  - 미디어/콘텐츠 링크(인스타/유튜브/블로그) — v1에선 링크만
- Placeholder:
  - 코치명: `{{COACH_NAME}}`
  - 소개: `{{COACH_BIO_SHORT}}`
  - 경력/자격: `{{COACH_CREDENTIALS}}`

### 5.4 프로그램(레슨 종류)
- 프로그램 카드 3종(권장):
  1) 필드 레슨
  2) 숏게임 레슨
  3) 외부 레슨(지역/장소 협의)
- 각 카드 포함 정보:
  - “이런 분께 추천” 2~3개
  - 진행 방식(대략)
  - 기대 결과(과장 금지, 현실적 표현)
  - CTA(“이 프로그램 상담하기” → 폼에서 관심 레슨 preselect 가능)
- Placeholder:
  - 프로그램 설명: `{{PROGRAM_FIELD_DESC}}`, `{{PROGRAM_SHORTGAME_DESC}}`, `{{PROGRAM_OUTSIDE_DESC}}`

### 5.5 진행 방식(프로세스)
- 프로세스 스텝(권장 4단계):
  1) 문의(전화/카카오/폼)
  2) 상담(목표/현재 실력/일정/장소)
  3) 레슨 진행(드릴/루틴/과제)
  4) 피드백/다음 단계 제안
- 요구사항: “무엇을 기대할 수 있는지”를 구체적으로(불안을 줄이기)

### 5.6 FAQ
- v1 최소 FAQ(권장):
  - 가격은 어떻게 되나요?
  - 준비물이 필요한가요?
  - 일정/장소는 어떻게 잡나요?
  - 환불/변경은 어떻게 되나요?
  - 초보도 가능한가요?
  - 법인/단체도 가능한가요?
- 답변은 “원칙 + 예외 + 다음 행동(문의)”로 짧고 명확하게 작성
- Placeholder:
  - 가격 정책: `{{PRICING_POLICY_SHORT}}` (v1에서 가격표를 공개하지 않을 경우 그 이유와 상담 유도 문구 포함)
  - 변경/환불 정책: `{{REFUND_POLICY_SHORT}}`

### 5.7 위치/시설(필수)
- 포함:
  - 주소/운영시간/주차 안내
  - 지도(네이버/카카오/구글 중 1개 링크 또는 임베드)
  - “지도 열기” CTA(이벤트 측정)
- Placeholder:
  - 주소: `{{ADDRESS}}`
  - 운영시간: `{{BUSINESS_HOURS}}`
  - 주차: `{{PARKING_INFO}}`
  - 지도 링크: `{{MAP_URL}}`

### 5.8 문의(CTA 섹션)
- 페이지 중간 1회 + 하단 1회(권장)
- 구성:
  - 문장: “지금 상황에 맞는 레슨을 상담받아보세요”
  - CTA 3개: 전화 / 카카오 / 문의 폼
- 법인/단체 문의는 작은 토글/서브카피로 분기(“단체/제휴 문의도 가능합니다”)

### 5.9 Sticky CTA (모바일 하단 고정)
- v1 필수(권장)
- 버튼 2~3개:
  - 전화(아이콘 + “전화”)
  - 카카오(“카카오 상담”)
  - 폼(“문의 남기기”)
- 접근성: 최소 터치 타깃 44px, 대비 준수

### 5.10 Footer
- 필수:
  - 상호/대표자/연락처(가능한 범위)
  - 개인정보처리방침 링크
  - (선택) SNS 링크
- Placeholder:
  - 사업자 정보: `{{BUSINESS_INFO}}`
  - 개인정보처리방침 URL: `{{PRIVACY_URL}}` (v1에서는 `/privacy` 권장)
